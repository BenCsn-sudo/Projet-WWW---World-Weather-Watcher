# Projet-WWW---World-Weather-Watcher
StationMeteo est un projet de station météo embarquée sur Arduino UNO mesurant la température, l’humidité, la pression, la luminosité et la position GPS. Les données sont enregistrées sur carte SD et indiquées par LED, avec gestion des modes via boutons et EEPROM, et également gestion des erreurs (débranchement, cohérence des valeurs...).
